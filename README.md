# jsramverk-me-app

## Project setup
```
npm install
```

## Install Vue
```
npm install -g @vue/cli
```

## Check Vue version
```
vue --version
```

# Update global Vue CLI paclage
```
npm update -g @vue/cli
```

```
vue create name_of_project
```

## Create new Vue project in GUI
```
vue ui
```
